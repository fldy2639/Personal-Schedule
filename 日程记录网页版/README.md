# 每日效率记录工具 v2.0.0

一个简单的单文件HTML应用，用于记录每日日程、任务、反思和苦强指数评估，并导出为Markdown格式，方便与Obsidian集成。

## 功能特点

- **双视图时间记录**：
  - **时间轴视图**：可视化时间块分布，类似甘特图
  - **编辑列表视图**：详细编辑每个时间段的开始/结束时间和活动
- **智能标签系统**：为每个时间块单独设置标签（工作、学习、休息等），颜色编码可视化
- **任务管理**：创建待办事项，标记完成状态
- **笔记与反思**：记录想法、收获、反思
- **苦强指数评估**：通过"变强"（主动增值）和"吃苦"（被动消耗）两个维度计算人生密度指数
- **模板系统**：预设模板功能已由苦强指数评估模块替代
- **数据持久化**：自动保存到浏览器本地存储
- **导出功能**：生成格式化的Markdown文件，完美兼容Obsidian
- **番茄钟计时器**：25分钟工作 + 5分钟休息计时，完成后自动记录时间块

## 使用方法

### 1. 基本使用
1. 双击打开 `index.html` 文件（推荐Chrome/Firefox浏览器）
2. 页面加载后会自动显示当前日期和时间
3. **时间记录**：
   - **时间轴视图**：点击"切换视图"进入时间轴，直观查看时间分布
   - **编辑视图**：在列表视图中编辑每个时间块的开始/结束时间和活动
   - **添加时间段**：点击"添加时间段"创建新的时间块
4. **智能标签**：
   - 先点击选中一个时间块（时间轴上的彩色条或编辑行）
   - 点击下方标签按钮（工作、学习、休息等）应用到选中的时间块
   - 标签会改变时间块颜色，便于视觉分类
5. **其他功能**：
   - **任务列表**：点击"添加任务"，输入任务内容
   - **笔记区域**：在文本框中记录想法和反思
   - **苦强指数评估**：
     - **变强评估**：为6个变强选项输入分数（0-100分）
     - **吃苦评估**：输入坐班、通勤、家务时长（小时）
     - **指数计算**：点击"查看人生密度指数"按钮计算最终指数
   - **模板系统**：模板功能已由苦强指数评估模块替代
6. **番茄钟功能**：
   - **开始计时**：点击番茄钟区域的"开始"按钮启动25分钟工作计时
   - **暂停/继续**：计时过程中可暂停和继续
   - **自动记录**：番茄钟完成后自动创建时间块记录
   - **调整时长**：点击工作/休息时间数字可自定义时长

### 2. 保存与导出
- **保存到本地**：点击"保存到本地"按钮，数据将保存到浏览器本地存储
- **导出Markdown**：点击"导出Markdown"按钮，生成 `.md` 文件并下载

### 3. 苦强指数评估

苦强指数评估通过"变强"（主动增值）和"吃苦"（被动消耗）两个维度计算人生密度指数：

1. **变强评估**：在人生密度评估模块中为6个变强选项输入分数（0-100分）：
   - 学到一个有用的知识点并付诸实践
   - 学会一个新方法or技术
   - 找到一个新客户
   - 认识一个厉害的人
   - 获得一笔超额回报（是时薪的3倍）
   - 主动向领导汇报一次
   - "以上都没有"：勾选此项将自动清零其他选项

2. **吃苦评估**：输入以下消耗性时间（小时）：
   - 坐班时长：日常工作办公时间
   - 通勤时长：上下班交通时间
   - 家务时长：家务劳动时间

3. **指数计算**：点击"查看人生密度指数"按钮，系统将自动计算：
   - 变强得分 = 所有选项分数总和
   - 总吃苦时长 = 坐班 + 通勤 + 家务时长
   - 人生密度指数 = 变强得分 ÷ 总吃苦时长

## 与Obsidian集成

### 步骤1：设置Obsidian
1. 在Obsidian中创建一个新的vault（知识库）
2. 记住vault的文件夹路径

### 步骤2：保存导出文件
1. 在效率记录工具中完成记录后，点击"导出Markdown"
2. 文件将下载到你的下载文件夹（文件名类似 `每日记录-2025-03-28.md`）

### 步骤3：组织文件结构（建议）
在Obsidian vault中创建以下文件夹结构：
```
MyVault/
├── Daily/                 # 每日日志
│   ├── 2025/
│   │   └── 03/
│   │       ├── 2025-03-27.md
│   │       └── 2025-03-28.md
├── Weekly/               # 周复盘
├── Monthly/              # 月复盘
└── Templates/            # 模板
```

### 步骤4：自动组织（手动方法）
1. 下载文件后，手动复制到 `MyVault/Daily/2025/03/` 文件夹
2. 在Obsidian中刷新文件列表即可看到

### 步骤5：利用Obsidian功能
- **双链笔记**：在笔记中链接其他笔记
- **图谱视图**：可视化笔记之间的关系
- **搜索功能**：快速查找历史记录
- **标签筛选**：使用 `#工作`、`#学习` 等标签

## 浏览器兼容性

- **推荐浏览器**：Firefox、Chrome、Edge
- **最低要求**：支持ES6、localStorage、Blob API的现代浏览器
- **已知限制**：
  - Firefox不支持原生的文件系统访问API，因此使用传统的文件下载方式
  - 某些旧版浏览器可能不支持 `type="time"` 输入框

## 数据安全与备份

- 所有数据存储在浏览器本地（localStorage）
- 建议定期导出Markdown文件作为备份
- 数据不会上传到任何服务器，完全隐私

## 高级使用技巧

### 1. 自定义模板
修改 `index.html` 文件中的模板内容：
1. 找到 `loadTemplate` 函数

3. 自定义时间块、任务和默认笔记

### 2. 修改样式
编辑 `index.html` 中的 `<style>` 部分，调整颜色、布局等。

### 3. 数据迁移
如需迁移数据，可以从浏览器的开发者工具中导出localStorage数据。

## 故障排除

### 问题：无法保存数据
- 确保浏览器支持localStorage
- 检查浏览器是否禁用了本地存储

### 问题：无法导出文件
- 确保浏览器支持Blob API
- 检查下载文件夹权限

### 问题：时间输入框不显示
- 使用文本输入代替，格式为 HH:MM（24小时制）
